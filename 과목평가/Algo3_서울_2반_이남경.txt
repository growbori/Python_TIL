 3-1. 이문제를해결하는데적합한자료구조의이름과그특성에대해간단히
서술하시오

스택 자료구조를 활용한다. 스택 자료구조는 선형 구조를 가지고 후입 선출 구조를 가진다는 특징이 있다.
따라서 문제를 풀 때 먼저 들어온 순서대로 출력되는 것이 아니라 맨 마지막에 들어온 요소부터 출력된다는 특징이 있다.
스택의 크기를 변경할 수 없다는 단점도 가지고 있다.

위의 문제를 풀기 위해선, 먼저 작성하고자 하는 문자열의 순서대로 스택에 요소들을 넣어준다.
만약 스택 맨 위에 있는 요소 (마지막으로 들어온 요소) 와 앞으로 들어올 요소의 문자가 같으면 기존에 들어있던 문자를
pop() 해주면 된다.

파이썬으로 작성해보면 아래와 같다.

arr = list(input())     # 문자열 전체 입력 받기

stack = []          # 빈 스택 생성하기

for i in range(len(arr)):   # 전체 범위에서
    if len(stack) == 0:     # 스택의 길이가 0이면
        stack.append(arr[i])    # 문자열에 있는 요소를 append()해준다.
    else:
        if stack[-1] == arr[i]:     # 스택의 맨 위에 요소 (마지막으로 들어온 요소) 와 투입할 문자열의 요소가 같으면
            stack.pop()     # 스택에 작성되어있는 요소를 pop() 해준다.
        else:
            stack.append()      # 위 조건에 해당되지 않으면 그냥 append()   해준다.

print(stack)
